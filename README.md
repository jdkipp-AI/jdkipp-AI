## Projects

### [InsightForge BI Dashboard](https://github.com/jdkipp-AI/InsightForge-BI-Dashboard)
An AI-powered business intelligence dashboard built with Streamlit, LangChain, and OpenAI
that combines interactive data visualization with conversational analytics and
retrieval-augmented generation.

### [MNIST GAN](https://github.com/jdkipp-AI/MNIST-GAN)
A Generative Adversarial Network trained on MNIST to generate handwritten digits from
random noise. Built with TensorFlow and Keras, with full documentation of the learning
progression from untrained noise through 100 epochs.

### [VAE Latent Space](https://github.com/jdkipp-AI/VAE-Latent-Space)
A Variational Autoencoder trained on MNIST with a 16-dimensional latent space, digit
morphing, and latent dimension traversal visualizations. Built with TensorFlow and Keras.

### [KippAI Chatbot](https://github.com/jdkipp-AI/KippAIChatbot)
A closed-domain RAG chatbot built with LangChain, ChromaDB, and Ollama that answers
questions from a local document knowledge base without relying on external APIs.
